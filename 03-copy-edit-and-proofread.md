# Editing Example – Wisteria Documentation

This example shows how unclear documentation can be improved for clarity and structure.

---

## Original version

Users may encounter issues when trying to access the Asteria platform due to incorrect credentials, expired sessions, or system-related problems and should ensure they are using the correct login details before attempting to reset their password or contact support.

---

## Edited version

Users may be unable to access the Asteria Platform for the following reasons:

- Incorrect login credentials
- Expired session
- Temporary system issues

### Recommended actions:
1. Verify your email and password
2. Reset your password if necessary
3. Refresh the page and try again

---

## Improvements made

- Broke long sentence into structured sections
- Separated causes from actions
- Improved scannability for end users
- Reduced cognitive load
